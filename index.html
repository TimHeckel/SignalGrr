    What is SignalGrr?

    SignalGrr is just an angrier, more impulsive SignalR proxy server. Why angrier? I'm not sure. Why more impulsive? Because it's so damn simple to spin up a hub using pure javascript. I made two simple jquery plugins using SignalGrr already: jquery.signalRamp.js - easy UI syncing and jquery.zyngaScroller.js - real-time kinetic scrolling.

    Can you handle it?

    Anyway, this whole client-side hub thing is so convenient, it made me want to shove it up on AppHarbor, so that's precisely what I did after upgrading the code to use the latest SignalR 1.0.0 javascript client and pulling the latest code from the SignalR repository.

    Now, if you're just wanting to schlep data between clients, stop worrying about the server side hub. Just write var conn = $.hubConnection("http://signalGrr.apphb.com") inside your javascript and you've got your entire hub!

    DISCLAIMER

    I know SignalR Alpha is nearing completion, so the api will likely change. I'll do my best to keep SignalGrr up to date with the latest javascript client and code.

    Client Usage

    This is inspired by David's original source:

    $(function () {
  var connection = $.hubConnection("http://signalgrr.apphb.com"),
    chat = connection.createProxy('chat');

    chat.on('foo', function (n) {
        console.log(n);
    });

    connection.start().done(function () {
        chat.invoke('foo', "If Jimmy Chitwood transfers to Terhune, we don't have a prayer this year.");
    });
});
